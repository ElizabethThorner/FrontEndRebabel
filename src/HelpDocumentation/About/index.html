About reBabel

Version: 1.0.0

reBabel is designed to simplify file format conversions between various language data files. It currently supports conversions between the following formats:

  • ELAN (.eaf)
  • Fieldworks Language Explorer (.flextext)
  • Universal Dependencies (.conllu)
  • Natural Language Processing (.txt)
