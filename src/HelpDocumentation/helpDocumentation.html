Importing

There are currently six file types supported for importing.

File Type Supported File Extension
Flextext .flextext
Conllu .conllu
NLP .txt
ELAN .eaf
SFM .sfm
Macula .xml

NLP Files

NLP file types have additional input parameters based on whether there is one combined file or if there are separate files.

Combined NLP File

A combined NLP file has a input parameter for a delimiter, by default /. Each word is separated from its part of speech by a delimiter, and the word/pos pairs are separated from each other by spaces. This would be an example of the file structure:

The/DET dog/NOUN barks/VERB ./PUNC
Separate NLP Files

Separate NLP files consist of one file containing the parts of speech and one file containing the words. This would be an example of the file structure for the part of speech file:

DET NOUN VERB PUNC

And this would be an example of the structure for the language file:

The dog barks .

Exporting

There are currently three file types supported for exporting.

File Type Supported File Extension
Flextext .flextext
Conllu .conllu
ELAN .eaf

Flextext Files

In order to export to the flextext format, the user has to provide the layers that will be present in the resulting flextext file out of "interlinear-text", "paragraph", "phrase", "word", and "morpheme".

ELAN Files

In order to export to the ELAN format, the user has to provide a .etf template file. The user also has the option to interpret time offsets as either seconds or milliseconds, with milliseconds as the default.
