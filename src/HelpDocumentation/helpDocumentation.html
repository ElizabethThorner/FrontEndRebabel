Importing

There are currently six file types supported for importing.

File Type Supported File Extension
Flextext .flextext
Conllu .conllu
NLP .txt
ELAN .elan
SFM .sfm
Macula .xml

NLP Files

NLP file types have additional input parameters based on whether they are a combined NLP file or separate NLP files.

Combined NLP Files

A combined NLP file has a input parameter for a delimiter. The delimiter by default is / and so a file using the default delimiter would be represented as

The/DET dog/NOUN barks/VERB ./PUNC
Separate NLP File

Separate NLP files consist of a part of speech file and a language file.
